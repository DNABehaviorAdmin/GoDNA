# Hugh Massie Portfolio

A premium, single‑page portfolio website for **Hugh Massie**, showcasing his role as Executive Chairman & Founder of DNA Behavior, his podcast, origin story, and ventures like GoDNA.

## Features
- Dark‑theme, glassmorphism design with smooth micro‑animations
- Custom logo and favicon integration
- Interactive background with particles and parallax
- SEO‑optimized meta tags, JSON‑LD schema, robots.txt & sitemap.xml
- Podcast section with Spotify & Apple Podcasts buttons
- Responsive layout built with Tailwind CSS (CDN) and vanilla JavaScript

## Project Structure
```
Hugh Massie/
├─ index.html        # Main page
├─ 404.html         # Custom 404 page
├─ styles.css       # Custom CSS & Tailwind overrides
├─ script.js        # Interaction logic
├─ assets/          # Images (portrait, logo, etc.)
├─ favicons/        # Favicon files
└─ ...
```

## Getting Started
1. Clone the repo:
   ```bash
   git clone <repo‑url>
   cd "Hugh Massie"
   ```
2. Open `index.html` in a browser (no build step required).
3. To develop locally, simply edit the HTML/CSS/JS files and refresh.

## Deployment
The site can be hosted on any static‑file host (GitHub Pages, Netlify, Vercel, etc.).
